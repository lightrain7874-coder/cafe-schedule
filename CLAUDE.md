## 프로젝트 개요

투썸플레이스 대평 펜트빌점 스텝(알바생) 이현우님이 만든 근무표 관리 웹앱.

**핵심 목적:** 단톡방에서 미제출자를 공개 호명하는 문화를 없애기 위해 제작.
점장님은 코드를 전혀 모르며, 현우님이 혼자 개발 및 운영 관리함.

---

## 아키텍처

```
[스텝/관리자 브라우저]
        ↓ fetch (5초 polling)
[GitHub Pages - index.html]
        ↓ fetch POST/GET
[Apps Script 웹앱 URL]
        ↓
[Google Sheets]
  - roster 탭      : 스텝 명단
  - submissions 탭 : 주차별 가능시간 제출 데이터
  - settings 탭    : 마감일 설정 (deadline_day/hour/min/on)
```

- **백엔드:** Google Sheets + Apps Script (`code.gs`)
- **프론트엔드:** `index.html` 단일 파일 (CSS/JS 인라인)
- **호스팅:** GitHub Pages (git push → 자동 배포)
- **사이트 URL:** https://bit.ly/dpy-schedule → https://lightrain7874-coder.github.io/cafe-schedule/
- **GitHub 저장소:** https://github.com/lightrain7874-coder/cafe-schedule

- **스텝(알바생):** 14명

## 구현된 기능 목록

### 스텝 입력 탭
- 이름 드롭다운 선택 (한 번 선택 시 localStorage에 자동 기억)
- 가능시간 그리드 (7일 × 3타임 = 21셀 클릭 토글)
- **퀵버튼 (2줄):**
  - 1줄: 전체 가능 / 전체 해제 / 평일만 (요일 셋팅형, 치환)
  - 2줄: 오픈만 / 미들만 / 마감만 (시간대 필터형)
    - 해당 시간대가 비어있으면 7일 전체 선택
    - 비어있지 않으면 해당 시간대만 남기고 나머지 해제
    - 조합 가능: 평일만 → 미들만 = 평일 미들 5개
- 메모 입력 (200자, 관리자에게 전달용)
- 지난 제출 불러오기 버튼 (그리드만 복원, 메모는 초기화)
- 이번 주 제출 완료 카드 (제출 후 상단에 표시)
- 마감 카운트다운 (미제출 상태일 때만 표시, 차분하게)

### 관리자 대시보드 탭
- 비밀번호 인증 (형식적, 클라이언트 코드에 노출됨)
- 주차 설정 (월요일만 선택 가능, 다른 요일 선택 시 알림)
- **마감일 설정 카드** (요일/시각 선택 → 시트 settings 탭에 저장 → 모든 스텝에게 반영)
- 통계 (제출완료/미제출/전체/제출률)
- 직원 명부 (추가/삭제, 시트 roster 탭과 연동)
- 미제출자 리마인더 (단톡 메시지 복사, 링크 포함)
- 알바생 메모 카드 (메모 있는 사람만 표시, 60자 미리보기 + 더보기)
- 통합 가용성 히트맵 (0 / 1-3 / 4-7 / 8-11 / 12+ 단계, 셀 클릭 시 모달)
- 히트맵 모달에서 가능자 이름 + 메모 아이콘 표시
- 새 제출 토스트 알림 (관리자 인증 후 활성화)
- 탭 제목에 미제출 인원 수 표시: "(3) 투썸 대평점 근무관리"
- CSV 내보내기 / 전체 요약 텍스트 복사 (메모 포함)
